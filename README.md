# 🛒 E-Kart Project

A basic e-commerce application that allows users to **place orders, manage deliveries, and view order details**.

---

## 📌 Features

* 🛍️ **Place Order**
  Users can select products and create new orders.

* 🚚 **Delivery Management**
  Track and manage delivery status of orders.

* 📦 **View Orders**
  Users can view all placed orders and their details.
